# <a name="supported-browsers-for-power-bi"></a>Power BI 支持的浏览器
在以下浏览器中运行 Power BI：

- Microsoft Edge
- Internet Explorer 11。 Internet Explorer 不支持某些高级功能，如世系视图。 有关详细信息，请参阅[数据世系（预览版）](service-data-lineage.md)。
- Chrome 桌面最新版本
- Safari Mac 最新版本
- Firefox 桌面最新版本。 Firefox 可以更改 Power BI 中使用的字体 

可用的所有平台都支持这些浏览器。
